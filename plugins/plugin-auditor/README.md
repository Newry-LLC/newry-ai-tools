# plugin-auditor

Use this skill whenever a Newry plugin or skill needs to be reviewed for quality — before shipping, after a major change, or when something feels off. Triggers on "audit this plugin," "review this skill," "check this SKILL.md," "is this well-designed," "look at the token efficiency," "review the implementation," or any request to evaluate whether a Newry plugin is ready. Audience is program maintainers, not consultants.
